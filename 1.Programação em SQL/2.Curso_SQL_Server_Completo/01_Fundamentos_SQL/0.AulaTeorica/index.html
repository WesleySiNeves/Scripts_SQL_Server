Versões do SQL Server

O SQL Server está disponível em diferentes edições, cada uma projetada para atender necessidades específicas, desde desenvolvimento até aplicações empresariais de grande escala.

Express

Versão gratuita ideal para desenvolvimento e aplicações pequenas

  • Até 10 GB de dados
  • Até 1 GB de RAM
  • Gratuito

Standard

Versão comercial para aplicações de médio porte

  • Sem limite de dados
  • Até 128 GB de RAM
  • Recursos completos

Enterprise

Versão premium com recursos avançados

  • Recursos ilimitados
  • Alta disponibilidade
  • Business Intelligence

Tipos de Dados no SQL Server

Tipos Numéricos

INT

Números inteiros de -2,147,483,648 a 2,147,483,647

DECLARE @idade INT = 25;

BIGINT

Números inteiros grandes de -9,223,372,036,854,775,808 a 9,223,372,036,854,775,807

DECLARE @id BIGINT = 123456789;

DECIMAL(p,s)

Números decimais com precisão fixa

DECLARE @preco DECIMAL(10,2) = 199.99;

FLOAT

Números de ponto flutuante com precisão aproximada

DECLARE @pi FLOAT = 3.14159;

Tipos de Texto

VARCHAR(n)

Texto variável com codificação ASCII

DECLARE @nome VARCHAR(50) = 'João';

NVARCHAR(n)

Texto variável com codificação Unicode

DECLARE @nome NVARCHAR(50) = N'José';

CHAR(n)

Texto fixo com codificação ASCII

DECLARE @codigo CHAR(5) = 'ABC01';

TEXT

Texto longo (depreciado, use VARCHAR(MAX))

DECLARE @desc VARCHAR(MAX) = 'Texto...';

Tipos de Data e Hora

DATE

Apenas data (YYYY-MM-DD)

DECLARE @data DATE = '2024-01-15';

DATETIME

Data e hora com precisão de milissegundos

DECLARE @dt DATETIME = '2024-01-15 14:30:00';

DATETIME2

Data e hora com maior precisão

DECLARE @dt2 DATETIME2 = '2024-01-15 14:30:00.123';

TIME

Apenas hora (HH:MM:SS)

DECLARE @hora TIME = '14:30:00';

Tipos Especiais

BIT

Valor booleano (0 ou 1)

DECLARE @ativo BIT = 1;

UNIQUEIDENTIFIER

Identificador único global (GUID)

DECLARE @id UNIQUEIDENTIFIER = NEWID();

XML

Dados estruturados em formato XML

DECLARE @xml XML = '<root><item/></root>';

JSON (2016+)

Dados em formato JSON (como NVARCHAR)

DECLARE @json NVARCHAR(MAX) = '{"nome":"João"}';
